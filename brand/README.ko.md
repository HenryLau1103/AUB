# AUB brand assets

Languages: [English](./README.md) · [繁體中文](./README.zh-Hant.md) · [简体中文](./README.zh-Hans.md) · [日本語](./README.ja.md) · **한국어**

Master mark 는 `aub-logo-mark.svg` 입니다. 네 개의 module 은 visual UI composition 을 나타내고, 중앙 node 는 coding agent 로 전달되는 structured handoff 를 나타냅니다.

## Files

- `aub-logo-mark.svg`: Primary full-color vector mark.
- `aub-logo-mark-flat.svg`: Print 와 compact use 용 flat-color vector.
- `aub-logo-mark-mono-dark.svg`: Light background 용 one-color mark.
- `aub-logo-mark-mono-light.svg`: Dark background 용 one-color mark.
- `favicon.svg`: Small-size simplified mark.
- `aub-app-icon.svg`: Rounded-square app icon master.
- `aub-maskable-icon.svg`: PWA safe-zone app icon master.
- PNG/ICO/ICNS files: Browser, PWA, Windows, macOS, Apple touch icon 용.

## Clear space and minimum size

Mark 주변에는 central node diameter 와 같은 clear space 를 유지합니다. 32px 미만에서는 simplified `favicon.svg` 를 사용합니다. Individual modules recolor, proportion 변경, central node 제거, busy image 위 배치는 피하세요.
