# AUB brand assets

Languages: [English](./README.md) · [繁體中文](./README.zh-Hant.md) · [简体中文](./README.zh-Hans.md) · **日本語** · [한국어](./README.ko.md)

Master mark は `aub-logo-mark.svg` です。4 つの module は visual UI composition を表し、中央 node は coding agent への structured handoff を表します。

## Files

- `aub-logo-mark.svg`: Primary full-color vector mark.
- `aub-logo-mark-flat.svg`: Print と compact use 向けの flat-color vector。
- `aub-logo-mark-mono-dark.svg`: Light background 用 one-color mark。
- `aub-logo-mark-mono-light.svg`: Dark background 用 one-color mark。
- `favicon.svg`: Small-size simplified mark。
- `aub-app-icon.svg`: Rounded-square app icon master。
- `aub-maskable-icon.svg`: PWA safe-zone app icon master。
- PNG/ICO/ICNS files: Browser、PWA、Windows、macOS、Apple touch icon 用。

## Clear space and minimum size

Mark の周囲には central node diameter と同じ clear space を確保します。32px 未満では simplified `favicon.svg` を使います。Individual modules の recolor、proportion の変更、central node の削除、busy image 上への配置は避けてください。
