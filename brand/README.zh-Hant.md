# AUB 品牌資產

語言： [English](./README.md) · **繁體中文** · [简体中文](./README.zh-Hans.md) · [日本語](./README.ja.md) · [한국어](./README.ko.md)

主標誌是 `aub-logo-mark.svg`。四個模組代表視覺 UI 組合，中央節點代表交付給 coding agent 的結構化 handoff。

## Files

- `aub-logo-mark.svg`：主要全彩向量標誌。
- `aub-logo-mark-flat.svg`：適合印刷與小尺寸使用的平面色標誌。
- `aub-logo-mark-mono-dark.svg`：淺色背景的一色標誌。
- `aub-logo-mark-mono-light.svg`：深色背景的一色標誌。
- `favicon.svg`：小尺寸簡化標誌。
- `aub-app-icon.svg`：圓角方形 app icon master。
- `aub-maskable-icon.svg`：PWA safe-zone app icon master。
- PNG/ICO/ICNS 檔案：瀏覽器、PWA、Windows、macOS 與 Apple touch icon 用途。

## Clear space and minimum size

標誌周圍保留等同中央節點直徑的 clear space。低於 32px 時使用簡化的 `favicon.svg`。不要重新上色單一模組、改變比例、移除中央節點，或把標誌放在過於複雜的圖片上。
