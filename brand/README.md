# AUB brand assets

The master mark is `aub-logo-mark.svg`. Its four modules represent visual UI
composition, while the central node represents the structured handoff to a
coding agent.

## Files

- `aub-logo-mark.svg`: primary full-color vector mark.
- `aub-logo-mark-flat.svg`: flat-color vector for print and compact use.
- `aub-logo-mark-mono-dark.svg`: one-color mark for light backgrounds.
- `aub-logo-mark-mono-light.svg`: one-color mark for dark backgrounds.
- `favicon.svg`: simplified small-size mark.
- `aub-app-icon.svg`: rounded-square app icon master.
- `aub-maskable-icon.svg`: PWA safe-zone app icon master.
- `favicon-16x16.png`, `favicon-32x32.png`, `favicon-48x48.png`: browser icons.
- `favicon.ico`: multi-size legacy browser icon.
- `aub-logo-mark-512.png`, `aub-logo-mark-1024.png`: transparent raster marks.
- `apple-touch-icon.png`: 180px Apple touch icon.
- `app-icon-192.png`, `app-icon-512.png`, `app-icon-1024.png`: app icons.
- `android-chrome-192x192.png`, `android-chrome-512x512.png`: conventional PWA icon names.
- `mstile-150x150.png`: Microsoft tile icon.
- `safari-pinned-tab.svg`: single-color Safari pinned-tab mark.
- `aub-app-icon.ico`, `aub-app-icon.icns`: Windows and macOS app icon bundles.
- `maskable-icon-512.png`: maskable PWA icon.

## Clear space and minimum size

Keep clear space equal to the central node diameter around the mark. Use the
simplified `favicon.svg` below 32px. Do not recolor individual modules, alter
their proportions, remove the central node, or place the mark over a visually
busy image.
