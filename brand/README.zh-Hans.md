# AUB 品牌资产

语言： [English](./README.md) · [繁體中文](./README.zh-Hant.md) · **简体中文** · [日本語](./README.ja.md) · [한국어](./README.ko.md)

主标志是 `aub-logo-mark.svg`。四个模块代表视觉 UI 组合，中央节点代表交付给 coding agent 的结构化 handoff。

## Files

- `aub-logo-mark.svg`：主要全彩向量标志。
- `aub-logo-mark-flat.svg`：适合印刷与小尺寸使用的平面色标志。
- `aub-logo-mark-mono-dark.svg`：浅色背景的一色标志。
- `aub-logo-mark-mono-light.svg`：深色背景的一色标志。
- `favicon.svg`：小尺寸简化标志。
- `aub-app-icon.svg`：圆角方形 app icon master。
- `aub-maskable-icon.svg`：PWA safe-zone app icon master。
- PNG/ICO/ICNS 文件：用于浏览器、PWA、Windows、macOS 与 Apple touch icon。

## Clear space and minimum size

标志周围保留等同中央节点直径的 clear space。低于 32px 时使用简化的 `favicon.svg`。不要重新上色单一模块、改变比例、移除中央节点，或把标志放在过于复杂的图片上。
